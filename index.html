Test APIs

APIs for testing with.

GET /houseid

Get the ID of the house.

Request query parameters:

  • housenumber - number of house
  • postcode - postcode of house

Response body contains the ID:

{"id": id of house}

GET /houseowner

Get details of a house.

Request query parameters:

  • houseid - valid house ID

Response body contains the house details:

{"id":"2468","owner":"David Seager","mortgageprovider":"Local Bank"}

PUT /houseowner

Update details of a house.

Request query parameters:

  • houseid - valid house ID

Request body:

{ "owner" : new owner, "mortgageprovider": new mortgage provider }

Response empty body with code 200

GET /postcodelookup

Get location details from a postcode.

Request query parameters:

  • postcode - postcode for example SO21 2JN. If not supplied returns all locations.

Response body contains the location. For all locations an array:

[
{"long":"1.39870","lat":"51.02655","building":"Bank A","street":"Hursley Park Road","area":"Hursley","town":"Winchester","county":"Hampshire","postcode":"SO21 2JN"},
...
]

For a specific location, the location:

{"long":"1.39870","lat":"51.02655","building":"Bank A","street":"Hursley Park Road","area":"Hursley","town":"Winchester","county":"Hampshire","postcode":"SO21 2JN"}
